What is a jupyter notebook?

A jupyter notebook combines instructions and text with editable, executable Python code.

In this course, we will be using jupyter notebooks to manage our programming exercises. Anytime we need to write, edit or execute code, we will do so through a jupyter notebook. Skeleton notebooks will be provided for every course exercise, so you won't need to write your own notebook from scratch. You are free to modify your notebook by adding text notes or other informational resources, and you will often be required to write, edit and/or execute code directly from your notebook.

Jupyter notebooks are not just for this course; they are a very popular mechanism for sharing annotated Python code amongst professionals in a wide variety of fields related to computer programming and data science. Once you learn to use jupyter notebooks effectively, you can use them to access existing content and share your code and data analyses with others.

Although jupyter notebooks are closely associated with the Python programming language and widely used in the Python programming community, jupyter notebooks can also be used to write, edit, execute and share code written in other programming languages. So don't think they are just a 'Python thing'.


What is google colaboratory?

Google's colab is a platform for running jupyter notebooks.

Google colaboratory or 'colab' is an online platform allowing anyone to access, modify, execute and share jupyter notebooks. Google colab connects your jupyter notebooks to high-performance computing resources using only a basic web browser, making it an incredibly powerful tool that can open the door to high-performance computing for users who don't necessarily want to become professional programmers.

If you dig a bit deeper, you'll find that google colab is basically just a specific instance of a platform called "jupyter hub", which is just a general server for sharing jupyter notebooks. So, google colab is a jupyter hub server, and there are other jupyter hub servers that are not google colab. Some of the nice things about colab is that it is free for public use, provides access to high-performance hardware and integrates with github and other repositories that support easy file sharing.
